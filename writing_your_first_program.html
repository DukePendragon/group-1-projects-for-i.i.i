Introduction

The first step in learning to program is writing a simple program to display "Hello, World!" on the screen. This is a great way to get started with any programming language.

Example in Different Languages

Below are examples of "Hello, World!" programs in various popular programming languages:

Python:
# This prints "Hello, World!" to the screen
print("Hello, World!")
JavaScript:
// This prints "Hello, World!" in the browser console
console.log("Hello, World!");
Java:
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
C++:
#include <iostream>
using namespace std;
int main() {
  cout << "Hello, World!" << endl;
  return 0;
}
C#:
using System;
class Program {
  static void Main(string[] args) {
    Console.WriteLine("Hello, World!");
  }
}

Steps to Run Your First Program

Here’s how you can run your "Hello, World!" program:

  • Python: Save the code as hello.py and run it using python hello.py.
  • JavaScript: Open your browser console (press F12 or Ctrl+Shift+I) and paste the code.
  • Java: Save the file as Main.java, compile it using javac Main.java, and run it with java Main.
  • C++: Save the file as hello.cpp, compile it using g++ hello.cpp -o hello, and run it with ./hello.
  • C#: Save the file as Program.cs, compile it using csc Program.cs, and run it with Program.exe.
