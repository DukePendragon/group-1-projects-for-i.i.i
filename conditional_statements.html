Learn how to use if-else and switch statements to make decisions in code.

Conditional statements are a key concept in programming that allow programs to make decisions based on certain conditions.

If-Else Statement

if (condition) {
  // code to run if the condition is true
} else {
  // code to run if the condition is false
}

Switch Statement

switch(expression) {
  case value1:
    // code to run if expression equals value1
    break;
  case value2:
    // code to run if expression equals value2
    break;
  default:
    // code to run if no case matches
}
