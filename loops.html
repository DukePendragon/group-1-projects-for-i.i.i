Loops

← Back to Homepage

Understand how to use for, while, and do-while loops to repeat tasks.

Loops are fundamental programming structures that allow you to repeat a block of code multiple times based on a condition. The most common types are for, while, and do-while loops.

For Loop

for (initialization; condition; increment) {
  // code to execute repeatedly
}

While Loop

while (condition) {
  // code to execute while condition is true
}

Do-While Loop

do {
  // code to execute at least once
} while (condition);

Examples

Here’s a summary of when to use each type of loop:

  • For Loop: Use when you know the exact number of iterations beforehand.
  • While Loop: Use when the number of iterations depends on a condition evaluated before each loop.
  • Do-While Loop: Use when the block of code must execute at least once, regardless of the condition.
