What are Variables?

Variables are like containers that store data values. In programming, you use variables to label and store information that your program can use or manipulate.

Common Data Types

Here are some of the most commonly used data types in programming:

  • String: Text values (e.g., "Hello, World!")
  • Integer: Whole numbers (e.g., 42, -3)
  • Float: Decimal numbers (e.g., 3.14, -0.5)
  • Boolean: True or False values

Example: Declaring Variables

Here is how you can declare variables in some common programming languages:

JavaScript:
let name = "John"; // String
let age = 25; // Integer
let isStudent = true; // Boolean
let pi = 3.14; // Float
Python:
name = "John" # String
age = 25 # Integer
is_student = True # Boolean
pi = 3.14 # Float
