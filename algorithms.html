What are Algorithms?

Algorithms are step-by-step instructions used to solve specific problems. They are fundamental to computer science and are used for tasks such as sorting data, searching through data, and solving complex problems efficiently.

Example: Sorting Algorithms

Sorting algorithms are used to rearrange data in a specific order. Here is an example of Bubble Sort:

JavaScript (Bubble Sort):
function bubbleSort(arr) {
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j < arr.length - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
}

let arr = [5, 3, 8, 4, 2];
bubbleSort(arr);
console.log(arr); // Output: [2, 3, 4, 5, 8]

Example: Searching Algorithms

Searching algorithms help you locate a specific item within a collection. Here’s an example of Binary Search:

JavaScript (Binary Search):
function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;

    while (left <= right) {
        let mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) {
            return mid;
        }
        if (arr[mid] < target) {
            left = mid + 1;
        }
        else {
            right = mid - 1;
        }
    }

    return -1; // Target not found
}

let arr = [1, 3, 5, 7, 9, 11];
let result = binarySearch(arr, 5);
console.log(result); // Output: 2

Advanced Problem-Solving Techniques

Advanced algorithms, such as Dynamic Programming, Greedy Algorithms, and Graph Algorithms, are used to solve more complex problems and optimize solutions.
