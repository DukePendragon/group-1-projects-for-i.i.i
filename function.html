What are Functions?

Functions are reusable blocks of code that perform specific tasks. They are essential for organizing your code, avoiding redundancy, and improving readability.

Example: Basic Function

Here is how to write a basic function in JavaScript:

JavaScript:
function greet(name) {
    return "Hello, " + name + "!";
}

let message = greet("John");
console.log(message); // Output: Hello, John!

Why Use Functions?

Functions help to break down large tasks into smaller, manageable pieces. They can also be reused multiple times, reducing code duplication and making your programs more efficient.

Function with Return Value

Here is how a function can return a value:

JavaScript:
function add(a, b) {
    return a + b;
}

let result = add(5, 10);
console.log(result); // Output: 15
